# 第 2 课：如何选取因子（从零开始的决策框架）

选取不是“找涨得多的指标”，而是一轮**假设检验**。

## Step A — 先分类，再挑选（避免随机刷指标）

经典大类（本仓库 Phase 1 覆盖）：

| 大类 | 例子 | 直觉 |
|---|---|---|
| 价值 Value | EP、BP | 便宜是否被补偿？ |
| 动量/反转 | 12-1 动量、1M 反转 | 趋势延续还是过度反应？ |
| 质量 Quality | ROE | 好公司是否持续赚取超额？ |
| 规模 Size | 对数市值 | 小盘是否有溢价？ |
| 风险/流动性 | 波动、换手 | 高风险/高交易是否反而更差？ |

**申请材料加分写法：** “我从文献大类出发构建候选集，而不是数据挖掘出偶然显著的怪因子。”

## Step B — 每个因子必须通过三道门

1. **故事门**：说不清机制 → 淘汰或降权  
2. **定义门**：计算是否稳健（极值、停牌、亏损股 PE）→ 去极值 + 标准化  
3. **证据门**：  
   - 月度 Rank IC 均值是否显著不为 0？  
   - ICIR（mean/std）是否尚可（教学经验阈值：> 0.3 可进入下一轮）？  
   - 五分层收益是否大致单调？  

本仓库脚本 `02_select_and_test_factors.py` 按此执行。

## Step C — 相关与冗余

两个 IC 都好但高度相关 → 合成时几乎只算一个。  
Phase 1 用等权合成；后续课会加相关矩阵与风险模型。

## Step D — A 股特有注意点（面试常问）

- 短期更常看到**反转**，中期动量不稳定  
- **市值、行业、流动性**会污染几乎所有因子 → 以后做中性化  
- 小盘因子容量与退市/微盘风格风险大  
- 交易成本与涨跌停约束会显著改变纸面结果
