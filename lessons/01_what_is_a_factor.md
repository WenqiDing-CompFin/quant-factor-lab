# 第 1 课：因子是什么？

## 一句话定义

**因子（factor）** = 一个可计算的股票特征，用来解释或预测**截面收益差异**（为什么有的股票下个月涨得更多）。

它不是“灵验的指标口号”，而是：

1. **定义清晰**的公式（任何人用同一数据应算出同一结果）
2. **有经济故事**（为什么市场会给这个特征定价）
3. **可检验**（IC、分层、回测能否稳定支持这个故事）

## 和“技术指标”的区别

| | 技术指标思维 | 因子研究思维 |
|---|---|---|
| 目标 | 猜下一根 K 线 | 解释一篮子股票的相对强弱 |
| 证据 | 盘感 / 个例 | 全样本统计（IC、分层、换手、成本） |
| 产出 | 买卖点 | 可合成的标准化得分 |

## 因子研究的标准流水线（本仓库按此实现）

```text
选故事 → 写定义 → 截面标准化 → Rank IC → 分层收益 → 合成 → 扣成本回测 → 可视化
```

对应代码：

| 步骤 | 位置 |
|---|---|
